Program 14-1

Generating Normally-Distributed Random Numbers

This applet demonstrates three algorithms for generating normally-distributed random numbers.

To run the demo:

  1. Choose an algorithm:
    • Central Limit Theorem
    • Polar
    • Ratio
  2. Press the Run button to animate the algorithm.
  3. Press the Pause button to pause the algorithm, and the Run button to resume it.
  4. To choose another algorithm, press the Pause button and the Reset button, then choose the algorithm.
