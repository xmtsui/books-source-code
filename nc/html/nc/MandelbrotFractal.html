Mandelbrot Set Fractal Image

This applet creates fractal images by plotting the Mandelbrot set derived from iterating complex functions of the form z^2 + c, varying c over the complex plane and then computing the orbit values for z = 0 + 0i. For each complex point c that is not in the set, the "escape velocity" of the orbit is represented by a shade of gray -- the more iterations, the darker the shade. For each point c that is in the set, its color is determined by the modulus of the final computed orbit value.

To run the demo:

  1. The initial fractal image is automatically generated.
  2. Because the fractal image is recursive, you can use the mouse to drag a rectangle around an area that you want to zoom into.
