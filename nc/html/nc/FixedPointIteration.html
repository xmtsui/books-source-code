Fixed-Point Iteration

This applet demonstrates finding a the root of a function f(x) by fixed-point iteration. For each function f(x), we can derive one or more iteration functions of the form x = g(x) by solving for x. We plot plot g(x) against the line x = y. Some of the functions g(x) will converge, and the rest will diverge.

To run the demo:

  1. Choose a convergent or a divergent function g(x).
  2. Drag the mouse along the x axis.
